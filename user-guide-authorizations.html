Guía de Usuario - Viajes TICA

Bienvenido a la guía de usuario para la pantalla Viajes TICA del sistema PLS. Esta guía está diseñada para ayudarte a utilizar y comprender de forma sencilla y directa las funcionalidades que ofrece esta pantalla.

Contenido

  • 1. Introducción
  • 2. Funciones principales
  • 3. Contacto y soporte

1. Introducción

Esta pantalla, ubicada en el submenú TICA Portones, permite filtrar y buscar información sobre los viajes registrados en el sistema. También ofrece detalles ampliados y la opción de actualizar los datos mostrados.

Consejo: Esta guía está diseñada para ser leída en orden, pero puedes saltar a las secciones que necesites usando el menú de contenido.

2. Funciones principales

1

Filtración de viajes

Para realizar esta función se debe hacer lo siguiente:

  1. Para esta función es necesario que el usuario maneje 2 campos en pantalla para mostrar la información deseada en pantalla siendo estos: Tipo de Viaje y Tipo Operación.
  2. Dependiendo de la opción seleccionada en el primer campo aparecerán diversos campos extra para realizar una búsqueda más específica.
    3. 1.1

    Filtración de viaje (Tipo de Viaje Contenedor)

    Para realizar esta función se debe de hacer lo siguiente:

    1. Se debe posicionar el cursor sobre la opción Tipo de Viaje y damos clic.
    2. Al momento de hacer clic se desplegarán diversas opciones para realizar la búsqueda.
    3. Una vez visualizando las opciones seleccionamos la opción Contenedor.
    4. Al momento de seleccionar está opción se agregará al lado del campo Tipo Operación el espacio para ingresar el número de contenedor.
    5. Previo a esto se debe seleccionar el tipo de operación Exportación o Importación.
    6. Por último, se debe agregar el número de contenedor deseado.
    7. Al dar clic en el botón Buscar se reflejará la información relacionada a la información seleccionada en los campos previos.
      8. Advertencia: El campo Contenedor es de carácter obligatoria.
    1.2

    Filtración de viaje (Tipo de Viaje Carga Suelta)

    Para realizar esta función se debe de hacer lo siguiente:

    1. Se debe posicionar el cursor sobre la opción Tipo de Viaje y damos clic.
    2. Al momento de hacer clic se desplegarán diversas opciones para realizar la búsqueda de forma más acertada.
    3. Una vez visualizando las opciones seleccionamos la opción Carga Suelta.
    4. Al momento de seleccionar está opción se agregarán diversos campos extra para realizar una búsqueda más específica.
    5. Antes de realizar la búsqueda se debe seleccionar el Tipo Operación.
    6. Continuando con los campos se debe de agregar el Número de viaje y el Número de arribo.
    8. Por último, se puede indicar que en el viaje hubo movimientos relacionados con autos dando clic en el checkbox Vehículo Rodado.
      9. .
    9. Una vez con toda la información seleccionada y agregada se puede dar clic en el botón Buscar para visualizar la información correspondiente.
      10. Advertencia: Los campos Número de viaje y Número de Arribo son de carácter obligatorias.
    1.3

    Filtración de viaje (Tipo de Viaje DUA)

    Para reralizar esta función se debe de hacer lo siguiente:

    1. Se debe posicionar el cursor sobre la opción Tipo de Viaje y damos clic.
    2. Al momento de dar clic sobre la opción anterior se desplegarán diversas para realizar la búsqueda de forma más acertada.
    3. Una vez visualizando las opciones seleccionamos la opción DUA.
    4. Al momento de seleccionar está opción se agregarán diversos campos extra para realizar una búsqueda más específica.
    5. Antes de realizar la búsqueda se debe seleccionar el Tipo Operación.
    6. Continuando con los campos extra se deben de agregar información oficial en estos para realizar la búsqueda.
    7. Por último, ya con toda la información seleccionada y agregada se puede dar clic en el botón Buscar para visualizar la información correspondiente.
      8. Advertencia: Los campos Número de Arribo, Código Aduana, Año y Número Correlativo son de carácter obligatoria.
2

Filtro mediante fechas

Para realizar esta función de manera satisfactoria se debe de realizar lo siguiente:

  1. Posicionamos el cursor sobre el campo Rango de fechas.
  2. Como se puede denotar en la imagen anterior el campo de fechas funciona con el orden Día/Mes/Año y es de carácter obligatoria para realizar cualquier búsqueda.
3

Búsqueda mediante filtro

Para realizar esta función se debe hacer lo siguiente:

  1. Colocar el cursor sobre el campo que muestra la leyenda Filtro.
  2. Una vez en este espacio se debe escribir el criterio de búsqueda deseado los datos se filtrarán con los resultados que coincidan con el criterio ingresado.
4

Función refrescar la información

Para lograr la función de refresco se debe hacer lo siguiente:

  1. Ubicar el cursor sobre el botón de refrescar y presionar sobre él.
  2. Al momento de dar clic se reflejará la información más reciente en el sistema que no haya sido proyectada en la tabla presente.
5

Información detallada de UT

Para lograr visualizar esta información detallada se debe de hacer lo siguiente:

  1. Ubicar el cursor sobre cualquiera de los íconos de Información que se encuentran en cada una de las filas de la tabla reflejada en pantalla.
  2. Al dar clic en esta opción se desplegará una ventana emergente con el siguiente aspecto.
6

Visualización de información pantalla Viajes TICA

Para brindar una experiencia más confortable a continuación se explicarán cada uno de los diversos campos que posee la tabla presente en pantalla.

  1. Número de contenedor: Identificador único del contenedor ante sistema y entes nacionales e internacionales.
  2. Unidad de transporte: Placa del vehículo que transporta la carga.
  3. Viaje: PREGUNTAR
  4. DUA: Documento único aduanero al cual se encuentra relacionado el viaje.
  5. Peso: Peso del contenedor en kilos totales.
  6. Tipo de carga: Tipo de carga que maneja el contenedor. Ejemplo: General, reefers, granel, etc.
  7. Autorizado por: Persona que autoriza el viaje.
  8. Fecha de autorización: Fecha y hora exacta en que se autorizó el viaje.
