Guía de Usuario - Categorías de Tarifa

Bienvenido a la guía de usuario para la pantalla Categorías de Tarifa del sistema PLS. Esta guía está diseñada para ayudarte a utilizar y comprender de forma sencilla y directa las funcionalidades que ofrece esta pantalla.

Contenido

  • 1. Introducción
  • 2. Funciones principales
  • 3. Contacto y soporte

1. Introducción

Ubicada en el submenú Facturación de Servicios, esta pantalla permite registrar, consultar, actualizar y gestionar categorías de tarifa, incluyendo funciones para editar o eliminar registros existentes y mantener la estructura tarifaria actualizada.

Consejo: Esta guía está diseñada para ser leída en orden, pero puedes saltar a las secciones que necesites usando el menú de contenido.

2. Funciones principales

1

Registrar nueva categoría

Para realizar un registro eficiente de una nueva categoría se debe hacer lo siguiente:

  1. Posicionar el cursor sobre el botón de Registrar ubicado en la parte superior izquierda de la pantalla.
  2. Una vez posicionado el cursor se debe dar clic sobre el botón el cual desplegará una ventana emergente con todos los campos requeridos para el registro.
  3. En la ventana de registro los campos que se deben de rellenar son los siguientes:
    1. Código: Identificador único de la categoría.
    2. Nombre: Nombre de la categoría a nivel de sistema.
    3. Descripción: Breve descripción de la categoría a nivel de sistema.
    4. Una vez completados todos los espacios se puede guardar la información dando clic en el botón Guardar para finalizar el proceso.
    5. En caso de que se desee cancelar la operación en cualquier momento se puede hacer clic en el botón Cancelar para abandonar el proceso.
2

Búsqueda por filtro

Para realizar la búsqueda por filtro de manera exitosa se debe de seguir los siguientes pasos:

  1. Posicionar el cursor sobre el campo Filtro ubicado en la parte superior izquierda de la tabla presente en la pantalla.
  2. Una vez posicionados en el campo se deben de ingresar los criterios de búsqueda donde si existen en el sistema alguna coincidencia con los datos registrados estos se mostrarán automáticamente en pantalla.
3

Función actualización de datos

Para realizar la actualización de datos se debe seguir los siguientes pasos:

  1. Posicionar el cursor sobre el botón ubicado en la parte superior izquierda de la tabla por debajo del botón de Filtro.
  2. Al momento de posicionar el cursor se debe dar clic sobre este para refrescar a la información más reciente existente en el sistema.
4

Opciones extra

Para realizar alguna de estas acciones se debe de seguir los siguientes pasos:

4.1

Editar categoría existente

Para editar una categoría existente se debe hacer lo siguiente:

  1. Localizar y posicionar el cursor sobre el botón de Opciones que se encuentra del lado izquierdo de todas las filas de la tabla.
  2. Una vez sobre el botón se debe dar clic para abrir una ventana emergente con las opciones correspondientes.
  3. La opción que se necesita en este caso es la de Editar que se encuentra posicionada de primera en la ventana emergente.
  4. Dentro de la opción Editar se encuentra una ventana emergente con la siguiente apariencia.
  5. Como se puede visualizar en la imagen anterior, se puede observar que se pueden editar todos los campos si es necesario.
4.2

Eliminar categoría existente

Para eliminar una categoría existente se debe hacer lo siguiente:

  1. Localizar y posicionar el cursor sobre el botón de Opciones que se encuentra del lado izquierdo de todas las filas de la tabla.
  2. Una vez sobre el botón se debe dar clic para abrir una ventana emergente con las opciones correspondientes.
  3. La opción que se necesita en este caso es la de Eliminar que se encuentra posicionada de segunda en la ventana emergente.
  4. Una vez seleccionada la opción Eliminar se desplegará una ventana emergente en la cual se debe confirmar la operación.
5

Visualización de información en pantalla Categorías de Tarifa

Para brindar una experiencia más sencilla para comprender la información presente en pantalla se explicarán a continuación los diversos campos que conforman la tabla de datos. De izquierda a derecha).

  1. Código: Identificador único de la categoría.
  2. Nombre: Nombre de la categoría a nivel de sistema.
  3. Descripción: Breve descripción de la categoría a nivel de sistema.
