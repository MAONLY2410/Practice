Guía de Usuario - Solicitud de Ingreso UT's

Bienvenido a la guía de usuario para la pantalla Solicitud de Ingreso UT's del sistema PLS. Esta guía está diseñada para ayudarte a utilizar y comprender de forma sencilla y directa las funcionalidades que ofrece esta pantalla.

1. Introducción

Ubicada dentro del submenú Procesos esta pantalla ofrece como función principal el ingresar los datos necesarios para realizar una solicitud de ingreso de UT's (Unidades de Transporte).

Consejo: Esta guía está diseñada para ser leída en orden, pero puedes saltar a las secciones que necesites usando el menú de contenido.

2. Funciones principales

1

Registro de solicitud para ingreso de UT's

Para realizar el registro correccto de la solicitud se deben de manejar los datos de la siguiente manera:

  1. Lo primero que se debe realizar es seleccionar la Naviera correspondiente que trabajará con la nave.
  2. Seguido al punto anterior se puede seleccionar el Arribo correspondiente a la nave.
  3. Por último, antes de lograr de manera satifactoria la solicitud de ingreso se debe seleccionar al Transportista que trabajará con la nave.
  4. Una vez realizados los pasos anteriores se pueden agregar archivos adjuntos si han de ser necesarios.
  5. Con todos los puntos listos y los archivos adjuntos agregados se puede proseguir a registrar la solicitud de ingreso.
Advertencia:Todos los campos de esta pantalla se encuentran marcados como obligatorios.
