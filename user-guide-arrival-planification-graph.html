Guía de Usuario - Gráfico de Planificación de Arribos

Bienvenido a la guía de usuario para la pantalla Gráfico de Planificación de Arribos del sistema PLS. Esta guía está diseñada para ayudarte a utilizar y comprender de forma sencilla y directa las funcionalidades que ofrece esta pantalla.

Contenido

  • 1. Introducción
  • 2. Funciones principales
  • 3. Contacto y soporte

1. Introducción

Ubicada dentro del submenú Planificación esta pantalla ofrece la posibilidad de visualizar de forma gráfica todos los arribos confirmados y registrados en el sistema a lo largo del año actual.

Consejo: Esta guía está diseñada para ser leída en orden, pero puedes saltar a las secciones que necesites usando el menú de contenido.

2. Funciones principales

1

Función actualizar datos

Para realizar la actualización de datos se debe seguir los siguientes pasos:

  1. Se debe localizar y posicionar el cursor sobre el botón de refresco, presente en el lado superior derecha de la pantalla.
  2. Una vez posicionado el cursor se debe dar clic sobre el botón el cual refrescará a la información más reciente registrada en el sistema.
2

Funciones complementarias para el manejo del gráfico

Para realizar las diversas funciones se deben de seguir las siguientes instrucciones:

2.1

Funcionalidades generales para el manejo del gráfico

Este apartado tiene diversas funcionalidades dentro del mismo por ende para poder manejarlos se debe realizar lo siguiente:

  1. Zoom + : Permite acercar el gráfico para mejorar la visualización de los datos presentes en el mismo.
  2. Zoom - : Permite alejar el gráfico para reducir la visualización de los datos presentes en el mismo y así brindar una vista mása genelarizada.
  3. Zoom en área seleccionada: Permite acercar el gráfico en un área previamente seleccionada con el cursor.
  4. Panning: Representado por el ícono de una mano, permite mover el gráfico de manera horizontal o verticalmente sin cambiar el nivel de zoom presente.
  5. Restablecer zoom: Representado por el ícono de una casa, permite restablecer el nivel de zoom al valor por defecto.
2.2

Función descargar gráfico

Para realizar esta función se debe hacer lo siguiente:

  1. Localizar y posicionar el cursor sobre el botón Menú.
  2. Una vez posicionado el cursor se debe dar clic en el botón para visualizar los diversos formatos de descarga que ofrece la opción.
  3. Ya con el formato decidido se puede dar clic sobre este para realizar la descarga del gráfico.
3

Mostrar o ocultar información del gráfico

Para realizar esta función se debe hacer lo siguiente:

  1. Colocar el cursor sobre cualquiera de los distintos elementos que componen el gráfico sea para ocultarlos o mostrarlos.
4

Visualización de información en pantalla Gráfico de Planificación de Arribos

Para brindar una experiencia más confortable, se explicarán los diversos campos que se encuentran presentes en la pantalla.

  1. # de arribo: Identificador único brindado por la empresa a las diversas embarcaciones que se compone por el año y un número de consecutivo.
  2. Fecha: Fecha en que se realizará el atraque.
  3. Nombre de la embarcación: Nombre de la embarcación registrada en el sistema.
