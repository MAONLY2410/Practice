Guía de Usuario - Tipos de Nave

Bienvenido a la guía de usuario para la pantalla Tipos de Nave del sistema PLS. Esta guía está diseñada para ayudarte a utilizar y comprender de forma sencilla y directa las funcionalidades que ofrece esta pantalla.

Contenido

  • 1. Introducción
  • 2. Funciones principales
  • 3. Contacto y soporte

1. Introducción

Dentro del submenú Almacenamiento de Datos podemos encontrar la opción para visualizar la pantalla Tipos de Nave donde podremos ver diferentes opciones entre modificar, eliminar, agregar y visualizar información de los diferentes tipos de buque registrados.

Consejo: Esta guía está diseñada para ser leída en orden, pero puedes saltar a las secciones que necesites usando el menú de contenido.

2. Funciones principales

1

Registrar tipo de nave

Para realizar la función principal se debe seguir los siguientes pasos:

  1. Ubicar el cursor sobre el botón de Acciones situado en la parte superior de la pantalla.
  2. Una vez ubicados sobre el botón Acciones damos clic derecho sobre él y se desplegará una ventana emergente con la opción de Registrar Tipo de Nave.
  3. Al dar clic en Registrar Tipo de Nave se desplegará otra ventana emergente con la opción de ingresar los diversos datos para una inserción nueva de un tipo de nave.
  4. Con lo anterior claramente explicado comenzamos con el registro de los datos para un nuevo tipo de nave.

    5. Registro Tipo de Nave

    1. Nombre: Es el nombre con el que se registrará el tipo de nave en el sistema.
    2. Descripción: Descripción breve sobre el tipo de nave y las características de este.
    3. Categoría de nave: Se debe seleccionar la categoría del tipo de nave que se desea agregar en base a las registradas en el sistema.
    4. Categoría COCATRAM: Se debe seleccionar la categoría COCATRAM en base a las registradas en el sistema para el tipo de nave que se desea agregar.
    5. Una vez completos los datos necesarios se puede confirmar la operación dando cli en el botón Guardar
    6. En caso de se desee cancelar la operación se puede hacer clic en el botón Cancelar
      7. Advertencia: El único campo no obligatorio en la ventana emergente es la Descripción, todos los demás son de índole obligatoria.
2

Función búsqueda por filtro

Para realizar la función búsqueda mediante filtro se debe realizar lo siguiente:

  1. Se debe posicionar sobre el campo de búsqueda el cual por defecto estará vacío con la leyenda de Filtro.
  2. Una vez que ubiquemos el campo se debe ingresar el dato el cual desea realizar la búsqueda y si existe una coincidencia se mostrará de manera automática en la tabla de datos.
3

Función refrescar información

Para refrescar la información los pasos a seguir son los siguientes:

  1. Se debe de colocar el cursor sobre el botón de refrescar y presionar sobre él.
  2. Una vez presionado el botón de refresco se mostrará la información más reciente agregada en el sistema sí ha de existir.
4

Función editar

Para realizar exitosamente la función de editar se debe seguir los siguientes pasos:

  1. Se debe colocar el cursor sobre el botón de Opciones situado en la parte izquierda de la tabla de datos en todas las filas.
  2. Una vez colocados sobre el botón Opciones se desplegará una ventana emergente con dos opciones Editar y Eliminar.
  3. Al dar clic en Editar se desplegará otra ventana emergente con la opción de editar algunos de los datos del tipo de nave seleccionado.
    4. Advertencia: En esta ventana emergente el único campo no modificable es Categoría de ahí los otros datos se pueden modificar.
5

Función eliminar

En este punto para realizar la función de eliminar se deben de seguir los siguientes pasos:

  1. Se debe de dar clic en la opción de Eliminar situada en la parte inferior de la ventana de opciones.
  2. Al momento de dar clic en la opción Eliminar se desplegará otra ventana emergente con la opción de confirmar la eliminación del tipo de nave seleccionado.
6

Visualización de la información en pantalla Tipos de Nave

Para tener una visión general más clara de la información proyectada en la pantalla se procederá a explicar de forma resumida los diferentes atributos que se encuentran en la tabla proyectada en la pantalla.

  1. Código: Es el código que identifica el tipo de nave en todas las pantallas del sistema relacionadas a esta.
  2. Nombre: Nombre con el que se registrará el tipo de nave en el sistema.
  3. Categoría: Categoría oficial del tipo de nave en el sistema.
  4. COCATRAM: Es la categoría COCATRAM correspondiente al tipo de nave en el sistema según la lista de COCATRAM.
