Guía de Usuario - Planificación de Atraques

Bienvenido a la guía de usuario para la pantalla Planificación de Atraques del sistema PLS. Esta guía está diseñada para ayudarte a utilizar y comprender de forma sencilla y directa las funcionalidades que ofrece esta pantalla.

Contenido

  • 1. Introducción
  • 2. Funciones principales
  • 3. Contacto y soporte

1. Introducción

Localizada dentro del submenú Planificación esta pantalla ofrece una vista general de la información relacionada a los buques, además de tener funciones de búsqueda, descarga y acciones extra que complementan el acceso a la información.

Consejo: Esta guía está diseñada para ser leída en orden, pero puedes saltar a las secciones que necesites usando el menú de contenido.

2. Funciones principales

1

Filtro de visualización por rango de fechas

Para realizar el filtro de visualización por rango de fechas se debe de seguir los siguientes pasos:

  1. Posicionar el cursor sobre el campo Rango de fechas ubicado en la parte superior izquierda de la pantalla.
  2. Una vez posicionado el cursor se puede dar clic en cualquiera de las fechas e incluso usar el ícono de calendario para seleccionar de una forma más sencilla el rango de fechas deseado.
  3. Una vez seleccionado el rango de fechas se puede dar clic en el botón de Filtrar para visualizar la información en ese espacio de tiempo.
2

Búsqueda mediante filtro

Para realizar la búsqueda mediante filtro de manera exitosa se debe de seguir los siguientes pasos:

  1. Posicionar el cursor sobre el campo Filtro ubicado en la parte inferior del campo Rango de fechas en la pantalla.
  2. Posicionados ahí se deben de escribir los criterios de búsqueda donde si existen coincidencias en el sistema se mostrarán automáticamente en pantalla.
3

Función de descarga

Para realizar la descarga de los datos en alguno de los diversos tipos existentes se deben de seguir los siguientes pasos:

  1. Localizar y posicionar el cursor sobre el botón Descargar el cual se encuentra ubicado en la parte superior derecha de la pantalla.
  2. Una vez localizado el botón, haz clic sobre él para ver los diferentes formatos disponibles para descargar la planificación de atraques.
  3. Una vez claro el formato deseado, se debe de dar clic sobre este para que seguido a eso se generé el archivo con la información correspondiente.
4

Función actualización de datos

Para realizar la actualización de datos se debe de seguir los siguientes pasos:

  1. Localizar y posicionar el cursor sobre el botón de refresco.
  2. Una vez localizado el botón, haz clic sobre él para actualizar la información a la más reciente del sistema si existe alguna.
5

Funciones extra

Para visualizar de manera correcta la información extra relacionada a los buques registrados se debe de hacer lo siguiente:

5.1

Visualizar información de la ventana

Para esta función se debe seguir los siguientes pasos:

  1. Localizar y posicionar el cursor sobre el botón de Opciones para desplegar las diversas funciones relacionadas al buque.
  2. Una vez localizado el botón, da clic sobre él para visualizar las diversas opciones.
  3. Una vez abierta la ventana de opciones, se debe dar clic en la opción Información de la Ventana para visualizar la información relacionada a la embarcación.
  4. Como se puede visualizar en la pantalla anterior se encuentra información importante como el puesto, calado, ETB, ETC y demás.
5.2

Visualizar información del arribo

Para esta función se dbee seguir los siguientes pasos:

  1. Localizar y posicionar el cursor sobre el botón de Opciones para desplegar las diversas funciones relacionadas al buque.
  2. Una vez localizado el botón, da clic sobre él para visualizar las diversas opciones.
  3. Una vez abierta la ventana de opciones, se debe dar clic en la opción Información del Arribo para visualizar la información relacionada a la embarcación.
  4. Como se muestra en la pantalla anterior, se puede visualizar información importante relacionada al atraque de la embarcación, como los datos generales, fechas y detalles del atraque.
5.3

Visualizar información de la nave

Para esta función los pasos a seguir son los siguientes:

  1. Localizar y posicionar el cursor sobre el botón de Opciones para desplegar las diversas funciones relacionadas al buque.
  2. Una vez localizado el botón, da clic sobre él para visualizar las diversas opciones.
  3. Una vez abierta la ventana de opciones, se debe dar clic en la opción Información de la Nave para visualizar la información relacionada a la embarcación.
  4. Como se muestra en la pantalla anterior, se puede visualizar un gran cantidad de información vital relacionada a la nave, como datos generales, información de navegación, etc.
5.4

Funcionalidad cancelar atraque

Para está funcionalidad se debe seguir los siguientes pasos:

  1. Localizar y posicionar el cursor sobre el botón de Opciones para desplegar las diversas funciones relacionadas al buque.
  2. Una vez localizado el botón, da clic sobre él para visualizar las diversas opciones.
  3. Un vez abierta la ventana de opciones, se debe dar clic en la opción Cancelar.
  4. Una vez seleccionada la opción, se debe de confirmar la operación.
6

Visualización de información en pantalla Planificación de Atraques

Para brindar una experiencia más confortable, a continuación se explicarán cada uno de los diversos campos que componen la tabla presente en la pantalla. (Yendo de izquierda a derecha).

  1. # de arribo: Identificador único brindado por la empresa a las diversas embarcaciones que se compone por el año y un número de consecutivo.
  2. Nave: Nombre de la embarcación registrado en el sistema.
  3. Identificador: Número de identificación a nivel internacional que posee la embarcación, por lo general se usa el IMO, pero se puede hacer uso de cualquiera siempre y cuando cumpla con las normas internacionales.
  4. Naviero: Nombre de la empresa que posee la embarcación.
  5. Año: Año en que se realizó el atraque.
  6. Semana: Número de semana en la que se realizó el atraque. (1 a 52)
  7. Puesto: Número de puesto o ventana en la que realizó el atraque.
  8. E.T.A: Reconocido como E.T.A (Estimated Time of Arrival) es la hora esperada de llegada de la embarcación a bahía.
  9. Hora estimada de atraque: Hora aproximada en la que se realizó el atraque.
  10. Hora estimada desatraque: Hora estimada para que el navio desatraque del puesto en puerto.
    11. E.T.D: (Estimated Time of Departure) es la hora estimada de salida de la embarcación.
  11. Calado proyectado: Profundidad máxima necesaria para que el barco pueda maniobrar libremente en el agua.
  12. Estado: Refleja de manera clara el estado en el cual se encuentra la embarcación al momento de visualizar la información en la pantalla
